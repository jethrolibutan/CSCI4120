## Quick Start
To get started, follow the steps below to ensure your environment is set up correctly:

### Prerequisites
Ensure that you have Python 3 installed on your machine. The following libraries are required to run the project:
- scipy
- pandas
- seaborn
- sklearn
- numpy
- matplotlib
- yellowbrick

### Installation
To install the required dependencies, run the following command:
sudo pip3 install -r requirements.txt
